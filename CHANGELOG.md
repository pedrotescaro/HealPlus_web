## [2.0.0] - 2024-12-XX

### Adicionado
- App mobile React Native com Expo
- Análise de feridas com IA (Google Gemini)
- Sistema de classificação etiológica de feridas
- Dashboard completo com estatísticas
- Sistema de relatórios PDF
- Chat assistente com IA
- Gestão completa de pacientes
- Agenda e compromissos
- Autenticação JWT
- Internacionalização (pt/en)

### Melhorado
- Interface responsiva mobile-first
- Performance e otimizações
- Segurança e validações

### Corrigido
- Bugs de autenticação
- Problemas de validação de formulários
- Erros de navegação

- Versão inicial do projeto
